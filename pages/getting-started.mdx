# Getting Started

To begin using the Boilerplate Templater extension, follow these steps:

1. Install the extension from the [Visual Studio Code Marketplace](https://marketplace.visualstudio.com/items?itemName=hanifmifta.boilerplatetemplater).

2. Initialize your workspace:
   - Press `Cmd+Shift+P` (or `Ctrl+Shift+P` on Windows/Linux) to open the command palette.
   - Search for `Templater: initialize` and select it.
   - This will create a `.templater` folder in the root directory of your workspace.
